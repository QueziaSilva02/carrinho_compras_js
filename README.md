# 🛒 Carrinho de Compras

Projeto desenvolvido em 2021, durante aulas de Linguagem e Desenvolvimento Web, do curso Técnico de Informática integrado ao Ensino Médio, no Instituto Federal de Caraguatatuba.

## 📝 Descrição do Projeto
Este projeto consiste em uma aplicação web interativa que simula um Carrinho de Compras. O foco principal da atividade foi exercitar a manipulação do DOM (Document Object Model), a lógica de programação com JavaScript e o fluxo de versionamento de código utilizando Git e GitHub.

### 🛠️ Tecnologias Utilizadas
- HTML5: Estruturação semântica da página.

- CSS3: Estilização e layout (fornecido na estrutura inicial).

- JavaScript (ES6+): Lógica de manipulação do DOM e cálculos.

- Git & GitHub: Versionamento e hospedagem do código
